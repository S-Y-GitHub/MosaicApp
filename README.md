# MosaicApp

## OverView

画像を合成してモザイク画像を生成します

## Requirement

* java 8
* javafx 8

## Usage

```
git clone -b https://github.com/S-Y-GitHub/MosaicApp
cd MosaicApp
javac MosaicApp.java
java MosaicApp
```

## Note

JavaFXが必要となるためLibericaJDK等の使用をお勧めします<br>
元画像の解像度や生成する画像の行・列数などにより時間を要する場合があります
